# Movie Finder 🎬

A React Native app to discover and explore trending movies using the TMDB API. 

## ✨ Features

- 🔍 Search movies
- 📈 View trending movies
- 📂 Browse by category
- 📄 View detailed movie information
- 🌙 Dark-themed UI

## 🔧 Tech Stack

- **React Native**
- **Expo**
- **TMDB API**
- **Tailwind CSS (via NativeWind)**
- **React Navigation**
